FAQ
  • MyWish is an open source project, you can check the source code on our GitHub, We guarantee security our services are 100% secure.
  • RAM/CPU/NET are resources of your account needed for interaction with EOS blockchain. You can setup it during the account creation if needed.
  • When you pay fees to create an EOS account, it means that you are paying for a certain amount of RAM which is required for the account to function.
  • It depends on how much RAM you need for your account, so the price differs but mainly it's 10 USD.
  • We generate a link to EOS blockchain once the account is created, so you will have a proof that now you own an EOS account.
  • EOS uses private and public key pairs for every account. If you don’t have public key we provide you with it and you need to save it in a safe place, However, MyWish is not saving any private pairs.
